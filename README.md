# Minesweeper

A classic Minesweeper game implementation.

## Website

This repository includes a static website describing the Minesweeper game, along with support and privacy policy pages required for iOS App Store submission.

The website is deployed to GitHub Pages.

### Pages

- [Home](https://kmanicka.github.io/Minesweeper/)
- [Support](https://kmanicka.github.io/Minesweeper/support.html)
- [Privacy Policy](https://kmanicka.github.io/Minesweeper/privacy.html)

### Deployment

To deploy the website to GitHub Pages:

1. Go to the repository settings on GitHub.
2. Navigate to "Pages" in the sidebar.
3. Under "Source", select "Deploy from a branch".
4. Choose the "main" branch and "/" (root) folder.
5. Click "Save".

The site will be available at `https://kmanicka.github.io/Minesweeper/`.
